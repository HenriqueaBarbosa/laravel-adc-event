## Sistema de gerenciamento de festas e eventos

Sistema desenvolvido em PHP/Laravel para gerenciar eventos, festas e participantes. Por meio dele podemos controlar a quantidade de participantes inscritos em um evento, quantos eventos foram criados por um anfitrião, adicionar e excluir a presença bem como o evento.

## Tecnologias utilizadas

1- PHP
2- Blade
3- Laravel
4- HTML
5- CSS
6- Javascript
7- MySql
8- Artisan
9- Jetstream
10- Livewire
11- Composer
12- Vite
13- Node

## Padrão de Projeto

Foi utilizado para esse projeto o padrão MVC(Model-View-Controller) e para autenticação, cadastramento e controle de usuários o auth();
